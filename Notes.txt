Site web
https://github.com/FranciscoAT/priv_bird

Terminal - site web:
	
	Aller dans le repertoire suivant:
		cd priv_bird/
		cd sample-site	
	
	Commencer le site web local
		npm start
	
	L'adresse du site web
		http://localhost:3000

	Pour voir le fichier P3P du site web
		http://localhost:3000/p3p.xml

--------------------------------------------------
Commandes utiles:
	
	Push
		En premier, il faut commit (local, seulement toi qui peut voir les modifications)		
			cd "ecrit le chemin du fichier que tu as modifier" (ex: cd priv_bird/sample-site)
			git commit index.js -m "commentaire ici pour ton commit" //j'ai modifier index.js
		Ensuite push
			git push

	Ajouter nouveau fichier
		cd "ecrit le chemin du fichier que tu as modifier" (ex: cd priv_bird/sample-site)
		git add form.pug	//ajouter le fichier form.pug
		git commit form.pug -m "commentaire ici pour ton commit" 
		git push
